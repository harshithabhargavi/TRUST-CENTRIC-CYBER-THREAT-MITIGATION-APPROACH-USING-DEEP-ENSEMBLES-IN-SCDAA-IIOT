CIMD-2024 ML artifacts folder
==============================

Place these 4 files here (from your notebook output or Kaggle download):

  1. preprocessor.joblib
  2. ensemble.joblib   (if you have "ensemble (1).joblib", rename it to ensemble.joblib)
  3. label_encoder.joblib
  4. metadata.json

They are produced when you run the full CIMD2024_ML_Pipeline.ipynb and save artifacts in Phase 9.

After copying the files here, reload the prediction page in the Django app.
